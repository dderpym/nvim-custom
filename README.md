# nvim-custom
my dotfiles for neovim

# how use
1. get [NvChad](https://nvchad.com/docs/quickstart/install) (updated files recently for version 2, if you're on version 1, you must be lame)
2. clone this directory into the custom directory
  ```sh
  git clone https://github.com/dderpym/nvim-custom ~/.config/nvim/lua/custom
  ```
3. open some files and stuff so the plugins do the thingy and install stuff
